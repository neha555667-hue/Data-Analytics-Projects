# 📊 Insurance Branch Performance & Sales Dashboard

## 📌 Project Overview

This project analyzes insurance branch performance, sales activities, policy metrics, payment status, and claim analysis using MySQL, Excel, Power BI, and Tableau. 
It provides interactive dashboards to monitor branch KPIs, track sales targets, and support data-driven business decisions.


# 🎯 Business Problem

Insurance organizations often struggle to monitor and compare branch-level performance due to fragmented data spread across multiple sources.

The objective of this project is to centralize business data, automate KPI tracking, and provide interactive dashboards to improve branch performance monitoring and strategic decision-making.


# 🛠 Tools Used

- Microsoft Excel
- MySQL
- Power BI
- Tableau


# 📂 Dataset

The dataset contains insurance business information including:

- Branch Performance
- Sales Targets & Achievements
- CRM Opportunities
- Meetings Data
- Invoice Reports
- Brokerage & Fees Records
- Policy Information
- Payment Status
- Claim Details



# 📈 Dashboard Preview

# 🔍 Key Insights

- Active, Lapsed, and Terminated policies are almost equally distributed, indicating opportunities to improve customer retention.
- Health Insurance has the highest number of policies among all insurance types.
- Payment Success Rate is **61.8%**, while failed payments highlight improvement opportunities in the collection process.
- Claim amounts are significantly higher than premium collections, emphasizing the need for better risk management.
- Interactive dashboards help identify top-performing and underperforming branches through real-time KPI monitoring.


# 💡 Recommendations

- Implement automated alerts for policies nearing lapse to improve customer retention.
- Strengthen payment collection workflows to reduce failed payment transactions.
- Monitor branch performance regularly using interactive dashboards.
- Align individual KPIs with branch-level targets to improve sales performance.
- Utilize dashboard insights to support strategic business decisions.
