# 📊 Crowdfunding Kickstarter Analytics

## 📌 Project Overview

This project analyzes Kickstarter crowdfunding campaigns to identify funding trends, campaign performance, success rates, and backer behavior using Excel, SQL, Power BI, and Tableau.


# 🎯 Business Problem

Crowdfunding platforms host thousands of projects, but many fail to reach their funding goals.

The objective of this project is to analyze historical Kickstarter campaign data and identify factors that contribute to project success.


# 🛠 Tools Used

- Microsoft Excel
- SQL
- Power BI
- Tableau


# 📂 Dataset

The dataset contains crowdfunding project information including:

- Project Category
- Funding Goal
- Amount Raised
- Number of Backers
- Launch Date
- Project Status

# 🔍 Key Insights

- Successful projects raised significantly more funding than failed projects.
- Projects with higher backer engagement had greater success rates.
- Lower funding goals resulted in higher campaign success.
- A small number of projects contributed a large portion of total funding.
- Campaign success varied over time.


# 💡 Recommendations

- Set realistic funding goals.
- Create engaging project descriptions.
- Maintain regular communication with backers.
- Use attractive visuals.
- Launch campaigns strategically.


# 🏆 Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Query Writing
- Dashboard Development
- Data Visualization
- Business Intelligence Reporting


# 📌 Conclusion

This project demonstrates how data analytics can help identify success factors in crowdfunding campaigns and support better decision-making for creators and stakeholders.
